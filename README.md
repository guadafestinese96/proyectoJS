# Proyecto JavaScript – Curso Coderhouse

Este proyecto fue desarrollado como parte del curso de **JavaScript en Coderhouse**.  
Su objetivo principal es aplicar conceptos fundamentales del lenguaje y construir funcionalidades dinámicas en el navegador.

## Tecnologías utilizadas
- **JavaScript (ES6+)**
- **HTML5**
- **CSS3**

## Funcionalidades
- Manipulación del **DOM** para actualizar contenido dinámicamente
- Uso de **eventos** para interacción con el usuario
- Validación básica de formularios
- Ejercicios prácticos de lógica y estructuras de control
- Implementación de pequeños módulos y scripts

## Objetivo académico
- Practicar fundamentos de programación en JavaScript
- Aprender a integrar JS con HTML y CSS
- Desarrollar interactividad en aplicaciones web
